# GWAS Catalog SV Signal Analysis

This directory contains scripts for analyzing structural variant (SV) signals in the GWAS Catalog, including LD-based tagging analysis, visualization, and Enformer model utilities.

## Directory Contents

### Scripts

- **SV_QTL_tag_by_GWASCatalog_lead_SNVs.R**: Identify SV QTLs that tag GWAS Catalog lead SNVs through LD analysis
- **vis.R**: Generate visualization for GWAS Catalog SV signal analysis
- **utils.py**: Utility functions for Enformer model operations, sequence extraction, and visualization
- **HN_DEL_03416.ipynb**: Jupyter notebook for specific SV analysis using Enformer model
