# CV-QTL Characteristic Analysis

This directory contains scripts for analyzing complex variant QTL (CV-QTL) characteristics, including SNP QTL comparison, cis-eQTL case analysis, and trans-QTL hotspot identification.

## Directory Structure

### SNV_QTL_comparison/
**Purpose**: Compare QTL effects with external QTL datasets
- `match_comp_func_utils.R`: Utility functions for allele matching and effect size correlation analysis

### cis_CV_eQTL_case/
**Purpose**: Analyze cis-eQTL characteristics by variant type
- `cis_eQTL_case.R`: Script for cis-eQTL analysis and visualization

### trans_QTL_hotspot/
**Purpose**: Identify and analyze trans-QTL hotspot regions
