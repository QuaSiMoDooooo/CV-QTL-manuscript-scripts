# Clinical Measurements GWAS Analysis

This directory contains scripts for analyzing genome-wide association studies (GWAS) on clinical measurements, including conditional analysis, fine mapping, and visualization.

## Directory Contents

### Scripts

- **Manhattan.R**: Generate Manhattan plots for clinical measurements GWAS analysis
- **conditional_analysis.smk**: Perform conditional analysis for clinical measurements GWAS using GCTA
- **fine_mapping.smk**: Perform fine mapping for clinical measurements GWAS using SuSiE
